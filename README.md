## 📌 Projetos em destaque

### 🦸 [Cypress Heroes](https://github.com/patricia-diosilva/cypress-heroes)
Projeto de automação de testes E2E com Cypress e TypeScript, utilizando **Page Object Model**.
Cobre cenários de login, listagem de heróis, criação, exclusão e controle de permissão por perfil de usuário (admin vs usuário comum).
